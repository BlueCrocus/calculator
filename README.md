# 🧮 calculator
html, css, js만을 사용하여 만든 간단한 사칙연산 계산기입니다.
## 🌟 주요 특징
* **기본 사칙연산** : 더하기, 뺴기, 곱하기, 나누기를 포함한 기본 계산 기능을 지원합니다.
* **계산과정 표시** : 현재 진행 중인 계산식 (`87 +`)을 메인 화면 상단에 작게 표시하여 사용자가 계산 과정을 볼 수 있습니다.
* **키보드 입력 지원** : 마우스 클릭 뿐만 아니라 키보드 (숫자, 연산자, Enter, Backspace)를 사용하여 사용할 수 있습니다.
* **오류 처리** : 0으로 나누는 경우 `Error` 메세지를 표시합니다.
---
## 🛠️ 기술 스택
이 프로젝트는 별도의 라이브러리나 프레임워크 없이 순수한 웹 기술만을 사용했습니다.
* **HTML5** : 기본적인 콘텐츠 구조 및 버튼 요소 정의
* **CSS3** : 스타일링 및 레이아웃
* **JS** : 계산 로직 및 DOM 조작

### ✨ CSS Grid를 활용한 레이아웃
계산기 버튼 영역은 **CSS Grid**를 사용하여 구현되었습니다.
* **`display: grid`** : 버튼 컨테이너에 적용하여 2차원 그리크 컨텍스트를 생성했습니다.
* **`grid-template-columns : repeat(4, 1fr)`** : 버튼들을 **4열**로 동일하게 나누어 배치했습니다.
* **`grid-column : span 2`** : 숫자 `0`버튼과 같이 크기가 다른 버튼에 적용하여 **2칸**을 차지하도록 설정하였습니다.

## 🚀 시작 방법
프로젝트를 로컬 환경에서 실행하는 방법은 매우 간단합니다.
1. **파일 다운로드** : 이 프로젝트의 모든 파일을 하나의 폴더에 다운로드합니다.
2.  **실행:** $\text{index.html}$ 파일을 웹 브라우저 (Chrome, Firefox 등)로 엽니다.

별도의 서버 환경 구축이나 의존성 설치 과정이 필요하지 않습니다.
